# DRACIN — CloudStream Plugin

Plugin CloudStream untuk menonton **29+ platform drama pendek** China & Korea sub Indo gratis dari [klikfilm.web.id](https://klikfilm.web.id).

## Cara Install

Tambahkan URL repository berikut di CloudStream:

```
https://raw.githubusercontent.com/Makairamei/DRACIN/builds/repo.json
```

## Fitur

- ✅ 29 platform drama pendek dalam 1 plugin
- ✅ Halaman utama dengan semua platform
- ✅ Lihat Semua drama per platform
- ✅ Search drama
- ✅ Daftar episode lengkap
- ✅ Streaming HLS (m3u8)
- ✅ Bahasa Indonesia

## Sumber

Semua konten diambil dari [klikfilm.web.id](https://klikfilm.web.id). Semua hak cipta milik masing-masing platform.
